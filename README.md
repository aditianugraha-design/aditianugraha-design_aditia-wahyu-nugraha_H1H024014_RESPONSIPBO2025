Aplikasi ini merupakan sistem simulasi berbasis web yang dibuat menggunakan PHP native dengan konsep Object Oriented Programming (OOP).  
Dalam aplikasi ini, trainer dapat melatih Pokémon 'Weepinbell' melalui beberapa jenis latihan seperti 'Attack, Defense, dan Speed', serta melihat perkembangan level dan HP Pokemon.  
Setiap sesi latihan akan disimpan dalam bentuk 'riwayat latihan', lengkap dengan waktu latihan.


## menggunakan
- PHP Native (tanpa framework)
- HTML
- CSS
- Server: XAMPP (Apache)


## cara masuknya 
1. Install dan jalankan *XAMPP*
2. Aktifkan *Apache*
3. Pindahkan folder project ke dalam: C:/xampp/htdocs/
4. Akses aplikasi melalui browser: http:/localhost/Aditia_H1H024014_ResponsiPBO2025
5. Gunakan menu:
- 'Mulai Latihan' untuk melatih Pokémon
- 'Riwayat Latihan' untuk melihat histori latihan



## Fitur Utamanya Mas
- Menampilkan informasi dasar Pokémon (nama, tipe, level, HP, jurus spesial)
- Melakukan latihan Pokemon dengan berbagai jenis latihan
- Menyimpan dan menampilkan riwayat latihan
- Upload gambar setiap sesi latihan
- Menggunakan konsep OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
